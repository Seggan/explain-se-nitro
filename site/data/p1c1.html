First, a bit of background information

SE Nitro is a webcomic created by Rydwolf Programs (known as “Redwolf Programs” in the comic) from Code Golf Stack Exchange (hereafter to be referred to as CGCC). It is mostly a comic version of in-jokes from the Stack Exchange and its chat. I will be linking to a lot of chat transcripts to prove my points. This explainer was created by Seggan from CGCC. The comic itself can be found here.

And now, on to the comic!

Person 1: How u fly?

Person 2: Must haz NITRO

SE Nitro is a joke on CGCC about a “premium” version of Stack Exchange that allows you to use Discord-style reactions in chat. Supposedly, Wheat Wizard had obtained it, after which math scat (the main antagonist of the comic) stole it from him. Apparently, SE Nitro can also give you flight. This panel might also be a reference to the antigravity XKCD.

Ginger: mathcat, no!

mathcat now decides to use his new powers for conquering the world, and is determined to destroy Ginger, who threatens to foil his conquest.

Ginger: Don’t make me use… GITHUB PERKS!

Github Perks is a related joke about the “only thing that can counter SE Nitro.” Ginger is the only one who has it.

Ginger: now… you die

Ginger attacks mathcat, apparently with a shield and nunchucks.

mathcat: or do I?

On the left is shown what is apparently a nether portal. Later, mathcat flees into there.
